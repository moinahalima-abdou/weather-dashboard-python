# weather-dashboard-python
Python weather application using OpenWeatherMap API to display real-time weather data by city input.

# 🌦️ Weather App (Python)

Simple weather application using OpenWeather API.

---

# 🚀 Features

- Search weather by city
- Display temperature, wind, humidity
- Save search history
- Web interface with Streamlit

---

# 🧱 Project Structure

- app.py → Web interface
- main.py → Console version
- services/ → API calls
- utils/ → history system
- data/ → saved history
